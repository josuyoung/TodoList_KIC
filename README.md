## 프로젝트

### 소개

Calendar와 TodoList를 결합하여 일정과 할일을 정리해주는 프로젝트

### 기간
- 24-01-23 ~ 23-01-25

### 핵심기능
- 본인이 원하는 날짜에 할 일을 적으면 달력에 표시와 날짜마다 할 일 출력

### 세부내용
- CRUD를 기반으로 만들어진 프로그램
- Calendar 내부 클랫드를 가져와 현재 시간을 기준으로 만들어짐

### 주요 객체
- DataPrinter 	
	- 메인 메서드에서 DataPrinter 객체 생성후 act()메서드를 이용해 CRUD의 기능을 입출력한다.

### 개발환경
- JAVA
